# AvgChat — Инструкция по запуску, деплою и сборке APK

Я подготовил для вас проект. В него добавлен код Android-приложения на **Kotlin**, который обертывает ваш сайт-мессенджер, а также настроены скрипты для деплоя сервера и фронтенда.

## 1. Деплой сервера (чтобы всё работало через интернет)

Самый простой способ бесплатно разместить сервер с базой данных и WebSockets — это **[Render.com](https://render.com/)**.

1. Создайте аккаунт на GitHub (если его нет) и загрузите туда эту папку как репозиторий.
2. Зарегистрируйтесь на [Render.com](https://render.com/) через GitHub.
3. Нажмите **New → Web Service**.
4. Выберите ваш репозиторий с `AvgChat`.
5. Укажите следующие настройки:
   - **Environment:** `Node`
   - **Build Command:** `npm run build`
   - **Start Command:** `npm start`
6. В разделе _Environment Variables_ (можете добавить по желанию):
   - `PORT`: `3001`
7. Нажмите **Create Web Service**.
8. Когда деплой завершится, вы получите публичную ссылку, например: `https://avgchat.onrender.com`.

## 2. Получение APK файла приложения (Android)

Вы можете получить APK двумя способами:

### Способ A: Через GitHub Actions (Не требует установки программ)

1. В файле `android/app/src/main/java/com/avgchat/app/MainActivity.kt` найдите строку 24:
   `private val APP_URL = "https://avgchat.onrender.com/"`
   _(Замените эту ссылку на ту, которую выдаст вам Render)_
2. Загрузите код на GitHub.
3. Перейдите во вкладку **Actions** в GitHub. Скажите «I understand my workflows, go ahead and enable them».
4. При каждом пуше в ветку `main`, GitHub Actions будет собирать приложение!
5. После завершения зелёной галочкой сборки "Android CI", в самом низу лога сборки вы найдете прикреплённый файл **AvgChat-APK**. Скачайте его на телефон и установите.

### Способ B: Через Android Studio (Локально)

1. Скачайте [Android Studio](https://developer.android.com/studio).
2. Нажмите **Open** и выберите папку `android` внутри AvgChat.
3. Также поменяйте ссылку в `MainActivity.kt` на вашу серверную.
4. Дождитесь конца синхронизации (Gradle Sync) - программа скачает всё нужное для сборки.
5. На верхней панели нажмите **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
6. В правом нижнем углу появится уведомление с путем до готового `.apk` файла.

Приложение настроено с разрешениями на микрофон, камеру и загрузку файлов, так что отправка аудио-сообщений, видео-кружочков и фото в Android будет работать корректно!
